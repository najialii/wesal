Organization Creation Test

Test: Organization Creation with Auto-Domain Generation
This test verifies that organizations can be created with automatic domain generation and admin user setup.
Step 1: Login as super admin at http://localhost:5175/login
Step 2: Use credentials: admin@wesaltech.com / password
Step 3: Navigate to Organizations → Create New Organization
Step 4: Enter organization name (e.g., "Test Company Inc")
Step 5: Observe auto-generated domain (e.g., "test-company-inc")
Step 6: Fill in admin details and select a plan
Step 7: Submit the form
Expected Result:
- Domain should auto-generate from organization name
- Form should submit successfully without domain validation errors
- Success message should show default password
- Organization should be created with admin user
- Subscription should be automatically created
Auto-Domain Generation Examples
Test these organization names to verify domain generation:
"Acme Corporation" → acme-corporation
"Tech Solutions LLC" → tech-solutions-llc
"ABC Company & Co." → abc-company-co
"123 Digital Agency" → 123-digital-agency

Troubleshooting
If you encounter issues:
1. Check that both frontend (port 5175) and backend (port 8000) servers are running
2. Verify database connection in Laravel
3. Check browser console for JavaScript errors
4. Check Laravel logs: backend/wesaltech/storage/logs/laravel.log
5. Ensure you're logged in as a super admin
